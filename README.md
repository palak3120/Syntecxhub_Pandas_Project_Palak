 Pandas CSV Reader & Basic Analysis

Project Overview

This project demonstrates how to use Python and Pandas to read CSV data, perform basic data analysis, filter records, and save results into a new CSV file.

Features

- Read CSV file using Pandas
- Display first and last rows
- Check data types
- Calculate mean, median, minimum, maximum, and count
- Filter employees by department
- Export filtered data to a new CSV file

Technologies Used

- Python
- Pandas

Files Included

- data.csv
- filtered_data.csv
- pandas_analysis.py
